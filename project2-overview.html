Properties of Wine Quality - Project Overview

Problem

The overall goal of the project was to analyze and evaluate the physicochemical properties of wine quality and identify any trends or patterns that points out the most influential factors.

Data

The dataset includes variables such as acidity, sugar, sulfur dioxidem pH, chlorides, alcohol, and much more. Data wrangling and preprocessing was done in order to produce the best results to later hand off to the data scientist.

Approach

The main platform used for the project was Google Colab, along with cleaning, transforming, and assessing the data so it can produce better results when being implemented into machine learning and visualizations.

Key Findings

  • Wine quality is influenced by Volatile Acidity & Alcohol Level.
  • Both variables show important relationships within the combined wine qualities as well as separately (Red & White).
  • The patterns/trends found will be able to contribute for future predictions to increase quality in the wine market.

Main Tools/Applications Used

  • Google Colab
  • Python
  • Pandas
  • Matplotlib
  • Plotly
