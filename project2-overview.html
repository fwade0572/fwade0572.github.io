Properties of Wine Quality - Project Overview

Problem

The overall goal of this project was to analyze the physicochemical properties of wine quality and identify any trends or patterns that could help in understanding which factors have the strongest relationship with wine quality.

Data

The dataset includes variables such as acidity, sugar, chlorides, sulfur dioxide, density, pH, alcohol, and wine quality. Before making any conclusions, data wrangling and preprocessing was conducted to establish the best possible results.

Approach

The main platform used for the project was Google Colab, along with performing exploratory data analysis, machine learning, and visualizations that could identify key trends related to the quality of wine.

Key Findings

  • Wine quality can be influenced by multiple physicochemical properties.
  • Alcohol content and acidity levels may show important relationships with wine quality.
  • The patterns revealed can help better understand which features contribute most to higher-quality wine.

Main Tools/Applications Used

  • Google Colab
  • Python
  • Pandas
  • Matplotlib
  • Plotly
