Crime in LA

Overview

This project analyzes crime data in Los Angeles to identify patterns, trends, and insights using data science techniques.

Approach

The dataset was explored using Python in Google Colab. Data cleaning, visualization, and analysis were performed to better understand crime distribution and trends.

Tools Used

  • Python
  • Pandas
  • Matplotlib / Plotly
  • Google Colab
