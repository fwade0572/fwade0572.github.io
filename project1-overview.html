Crime in LA - Project Overview

Problem

The goal of this project was to analyze crime data in Los Angeles to identify trends, patterns, and potential insights that could help understand crime distribution.

Data

The dataset included crime records such as location, type of crime, date, and other related variables. Data cleaning and preprocessing were required before analysis.

Approach

Using Python in Google Colab, I performed exploratory data analysis, created visualizations, and identified key trends in the dataset.

Key Findings

  • Certain crime types occur more frequently in specific areas
  • Crime trends vary based on time and location
  • Patterns can help inform prevention strategies

Tools Used

  • Python
  • Pandas
  • Matplotlib / Plotly
  • Google Colab
