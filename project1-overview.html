Crime in LA - Project Overview

Problem

The overall goal of this project was to analyze the crime in LA and identify any trends or patterns that could help in understanding where the most amount of crimes were, along with their geographical location. From there, solutions such as increasing security can be made.

Data

The dataset includes variables such as the type of crime committed, date, location, and much more. Before making any conclusions, data wranggling and preprocessing was conducted to establish the best possible results.

Approach

The main platform used for the project was Google Colab, along with performing exploratory data analysis, machine learning, and visualizations that could identify key trends that need the most attention.

Key Findings

  • There is a common type of crime given a certain area dependent on its specifications.
  • Crime trends tends to be most linked to the time and location in which they are committed.
  • The patterns revealed can help to apply solutions for both strategy and safety.

Main Tools/Applications Used

  • Google Colab
  • Python
  • Pandas
  • Matplotlib
  • Plotly
